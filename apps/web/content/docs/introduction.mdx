# Introduction

Welcome to **KitWeaver**! This library provides a collection of copy-paste components for your Next.js applications.

## Features

- **Copy & Paste**: No npm install required for components.
- **Customizable**: Built with Tailwind CSS.
- **Accessible**: Powered by Radix UI.

## Getting Started

To use a component, simply browse the library, copy the code, and paste it into your project.

```bash
npx shadcn@latest init
```

Check out the [Button](/docs/components/button) component to get started.
