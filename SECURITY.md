# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability:

1.  **Do NOT open a public issue.**
2.  Email `security@kitweaver.com` (or the maintainer's email).
3.  Include details:
    - Type of vulnerability.
    - Steps to reproduce.
    - Impact assessment.

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Policies

- **Data Privacy**: We comply with GDPR. User data requests are handled via support.
- **Auth**: We use industry-standard providers (Clerk/Supabase). We do not store passwords.
- **Payments**: All payment data is handled by Stripe. We do not touch credit card numbers.
